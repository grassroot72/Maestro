ASCII was the first character encoding standard (also called character set). Because ANSI and ISO-8859-1 were so limited, HTML 4 also supported UTF-8. UTF-8 (Unicode) covers almost all of the characters and symbols in the world. The default character set for HTML5 is UTF-8.
